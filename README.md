# Personal Portfolio Website

## Overview
This project is a personal portfolio website built using HTML and CSS. 
It was designed in Figma and implemented as a multi-page website with a single-page 
scrolling experience on the Home page.

## Features
- 5-page website (Home, About, Projects, Resume, Contact)
- Smooth scrolling navigation on Home page
- Project cards with links to live demos and GitHub
- Styled contact form
- Consistent UI based on Figma design

## Technologies Used
- HTML5
- CSS3
- Figma
- Git & GitHub
